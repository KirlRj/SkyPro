# Проект "Обучение SkyPro"

## Описание:

Проект "Обучение SkyPro" это выполнение домашнего задания по курсу "Python разработчик"

На текущий момент в проекте есть следующая функциональность:
1. get_mask_card_number - функция возвращает маску номера карты по правилу XXXX XX** **** XXXX.
2. get_mask_account - функция возвращает маску номера счета по правилу **XXXX.
3. mask_account_card - функция возвращает маску. Для карты XXXX XX** **** XXXX. Для счета **XXXX.
4. get_date - функция преобразует дату в формат 'DD.MM.YYYY.
5. filter_by_state - функция возвращает новый список словарей, содержащие ключ state == входному значению.
6.  sort_by_date - функция сортирует список словарей по дате. По умолчанию - убывание.
